# これは

WebRTCを使ったビデオ通話アプリです。

以下の機能があります。

- Communication Mode（P2P, MESH, SFU）の選択
- 送受信、送信のみ、受信のみ の選択
- 映像、音声コーデックの指定
- カメラの解像度、フレームレートの指定
- カメラ、マイク、スピーカーのデバイス切り替え
- ミュート（カメラ、マイク、スピーカー）、ボリューム調整（スピーカー、Chromeのみ）
- 画面共有
- ネットワーク帯域の指定
- PWAインストール
- 音声認識（Chromeのみ）
